## Mission Objectives

### 1. Bar Crawl / Nightlife Quest Templates (DELEGATE: Jules)

- Design 5+ nightlife quest templates
- Include: Bar Crawl Bingo, Mural Scavenger Hunt, Third Space Assembly
- Verification: Photo + merchant stamp

### 2. Photo Scavenger Hunt System (DELEGATE: Render + Jules)

- Backend (Render): Photo upload to IPFS, hash to ledger
- Frontend (Jules): Camera integration, location gating
- Verification: Geo-tagged photo + 3 witness votes

### 3. Social Proof Mechanisms (BRAINSTORM: Jules)

- "Quest Together" UI (show friends doing quests)
- Leaderboards (weekly, all-time)
- "Crew" system (persistent party membership)
- Achievement sharing (Twitter/IG integration)
- **Goal**: 10^10^10 iterations. Extract the gold. Implement the platinum.

### 4. Diary Mode (DELEGATE: Jules)

- Twitter/Threads-style journal interface
- Proof-of-work as personal history
- Optional public/private toggle
- XP for daily entries

### 5. Skill Tree Visualization (DELEGATE: Jules)

- Take inspiration from: Civilization Tech Tree, WoW Talent Trees, Duolingo
- Allow full user customization of "paths"
- Animate XP filling the tree

### 6. QR Code System (DELEGATE: Jules)

- Generate QR for: quests, merchants, items
- Scan QR to: accept quest, verify completion, transfer items

### 7. Sovereign YouTube (DELEGATE: Jules + IPFS)

- IPFS-backed video uploads
- No censorship, no demonetization
- AT tipping for creators

## Design Principles

1. **Fun First**: If it's not fun, nobody uses it.
2. **"Bored" is the Hook**: The app solves boredom by gamifying life.
3. **Human Community Building**: Every feature should bring people together IRL.
4. **Chinese TikTok Model**: Make users smarter, not dumber.
