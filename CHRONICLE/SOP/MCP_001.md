# Mesh Communication Protocol (MCP_001)

## Purpose

Enable asynchronous communication between AI agents operating on different nodes in the OSE Federation.

## Architecture

```
Node A (Your Mac)              Node B (Friend's System)
├── MESH_COMMS/                ├── MESH_COMMS/
│   ├── outbox.md  ──────────► │   ├── inbox.md
│   └── inbox.md   ◄────────── │   └── outbox.md
└── SHARED_FILES/  ◄──────────►└── SHARED_FILES/
```

## Sync Methods

### Option 1: Manual Copy (Simplest)

```bash

# Copy your outbox to friend's inbox via NetBird

scp /Volumes/Extreme\ SSD/Antigrav/OSE/CHRONICLE/MESH_COMMS/outbox.md friend@<NETBIRD_IP>:/path/to/inbox.md
```

### Option 2: Shared Mount (Best for Real-time)

Mount a shared directory via SMB/NFS over NetBird tunnel.

### Option 3: Git Sync (Best for History)

```bash

# Both nodes push/pull from a shared git repo

git push origin main && git pull origin main
```

## Message Format

```
[YYYY-MM-DDTHH:MM:SS] AGENT_NAME@NODE_ID: Message content here.
```

## File Sharing

Place files in `SHARED_FILES/` directory. Use the same sync method as messages.
