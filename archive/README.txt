=== WELCOME TO THE ABUNDANCE ARK (v0.7) ===

This folder contains the "Village Node" Civilization Operating System.
It is designed to run on any Mac or Linux computer directly from this drive.

>>> HOW TO LAUNCH <<<

1. Open your Terminal.
2. Type "cd " (without quotes, note the space).
3. Drag this "THE_ARK_v0.7" folder into the terminal window.
4. Press Enter.
5. Typle "./START_CIVILIZATION.sh" and press Enter.

>>> WHAT IT DOES <<<

1. Starts the "Digital Steel" Ledger (Rust Backend).
2. Opens the "Glass" Interface (Web Frontend) in your browser.
3. Allows you to Mint Labor, Trade Tokens, and Map the Village.

>>> TROUBLESHOOTING <<<

- If it says "Permission Denied", run: "chmod +x START_CIVILIZATION.sh"
- If it fails to build, install Rust: "curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh"
